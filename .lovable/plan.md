

# 🚀 TNS Lab Hub — Portail Interactif de TPs

Un site vitrine spectaculaire pour tes TPs de Traitement Numérique du Signal, avec un design glassmorphism futuriste et des animations impressionnantes.

---

## Page d'accueil — Landing épique

- **Hero animé** avec un titre stylisé "TNS Lab" et une animation de forme d'onde sinusoïdale en arrière-plan
- Effet **glassmorphism** (panneaux semi-transparents avec flou) sur fond gradient animé (violet/bleu/cyan)
- Particules ou lignes animées en background pour un effet "tech/signal"
- Infos : Université d'Oran, 3ème année Ingénieur, Sécurité Informatique
- Bouton d'accès au TP1 avec animation hover spectaculaire

---

## Page TP1 — Généralités sur les Signaux

### Navigation par exercice
- Barre latérale ou tabs avec les 4 exercices, navigation fluide avec transitions de page animées

### Exercice 1 — Visualisation des signaux
- **Graphiques interactifs** (recharts) pour chaque signal (x1 à x12) : zoom, hover avec valeurs, toggle affichage
- Sélecteur de signal avec effet glassmorphism
- Bloc de **code Python** avec coloration syntaxique pour chaque signal
- Classification énergétique affichée visuellement (barres/badges)

### Exercice 2 — Fonctions et convolution
- Visualisation des signaux x(t), y(t), z(t), w(t) avec graphiques interactifs
- Code Python avec coloration syntaxique
- Explication des relations U(t) ↔ sgn(t)

### Exercice 3 — Signal graphique x14(t)
- Reproduction interactive du signal graphique donné dans le TP
- Analyse visuelle avec annotations

### Exercice 4 — Décomposition paire/impaire
- Graphiques côte à côte : signal original, partie paire, partie impaire
- Code Python source affiché

---

## Design & Animations

- **Thème sombre** avec accents néon (cyan, violet, magenta)
- **Glassmorphism** partout : cartes semi-transparentes avec backdrop-blur
- **Transitions de page** animées (fade + slide)
- **Animations d'entrée** des éléments au scroll (staggered fade-in)
- **Hover effects** spectaculaires sur les cartes et boutons
- **Gradient animé** en background qui bouge lentement
- Responsive mobile

---

## Structure extensible

- Architecture modulaire pour ajouter facilement les prochains TPs (TP2, TP3...)
- Page d'index des TPs avec cards animées
- Chaque TP est une section indépendante
